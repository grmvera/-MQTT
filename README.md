# Simulación IoT con MQTT

## Descripción
Este proyecto simula un sistema IoT utilizando MQTT. Incluye un sensor de luz y un interruptor que enciende/apaga las luces según la intensidad de luz.

## Requisitos
- Python 3
- Librería `paho-mqtt`
- Mosquitto Broker MQTT
